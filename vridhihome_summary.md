## 🏗️ Technical Architecture Stack

You have built a robust, modern technology stack:

### 1. Frontend (The Interface)
* **Framework:** React.js powered by Vite.
* **Styling:** Custom Vanilla CSS for maximum control and dynamic animations.
* **Hosting Provider:** Vercel
* **Core Role:** Handles the beautiful user interface, the intake engine form, and displays the personalized Enhancement Blueprint dashboards.

### 2. Backend (The Engine)
* **Framework:** Node.js with Express.js
* **Security:** JWT (JSON Web Tokens) for authentication, and `bcrypt` for secure, encrypted password hashing.
* **Hosting Provider:** Render
* **Core Role:** Validates logins, calculates AI Ghar-Scores, processes intake forms, and orchestrates database transactions securely over the internet.

### 3. Database (The Memory)
* **System:** PostgreSQL
* **Hosting Provider:** Neon Serverless DB
* **Core Role:** Securely stores your users, hashed passwords, property profiles, and master upgrade recommendations. It is hosted independently so your Render server can read and write to it anytime.
